Introduction

Multix is a responsive, user-friendly and easy-to-use content management system that is built with the popular Codeigniter framework. Using this CMS, you can easily build a nice, modern and clean website that have almost all the pages like team member, about, service, portfolio, news, event, contact etc. You can build education website, lawyer website, gym and fitness website etc. what you want to do.

This CMS is developed using the popular PHP framework Codeigniter. So, it can be easily customized by other programmers. Developers will be able to understand the source code easily and can modify the database and files if needed.

This script/CMS is strong against most vulnerabilities on the web like SQL injection attack, XSS attack etc. Also admin level security is implemented whereever necessary.

Requirements

Following modules must be activated on your server:

  1. PHP version: 7.0 or higher
  2. PDO Extension
  3. fileinfo Module
  4. GD Library
  5. cURL
  6. allow_url_fopen
  7. Mbstring

Installation

In order to install the CMS, please follow these steps:

  1. Unzip the file you download from Envato.
  2. Search for the folder cms in the main script folder that you have downloaded.
  3. Now upload the cms folder into your server using FTP.
  4. Alternately you can zip the cms folder and upload it to server via cpanel. Then extract this and use.
  5. You can use our cms into root domain or subdomain.
  6. Become sure that you have uploaded the .htaccess file correctly. When you will extract the zip file staying on server via cpanel, the .htaccess file may not be extracted properly in the server and then the script will show error. So after extracting, you may have to manually upload the .htaccess file into the server that comes with the script file.
  7. Create mysql database using cpanel or your hosting provider's system. Then import the multix.sql file (that you got into the script's database folder) into your created database.
  8. Now you will have to setup the config and database file. If you are familier with codeigniter, you should know about it. But don't worry! We will tell step by step.
  9. Go to the cms folder, then application and then config folder.
  10. You will get two files config.php and database.php

  11. Open up these files with any text editor (for example: notepad++).

Admin Panel Login

In order to login to the admin panel, go to the url below:

http://www.yourwebsite.com/admin

Here "yourwebsite" is your main domain name. Just after your main url, you have to put "admin" and that's it.

Support

Support Process (if support is on)

  1. First look at the navigation menu on the left side of the documentation. You will get all important topics here.
  2. Search the entire document pressing "ctrl+F", enter your query string and press "Search".
  3. If you still do not get the answer of your question, then contact us via codecanyon and always give your live URL when asking for help.
  4. When asking for help, please provide the details and also give some screenshots if possible.

Why we do not give support through codecanyon comment section?

  1. Do not ask for support in the comment section of codecanyon. Because our codecanyon account access is restricted and support team will not be able to reply. In that case, support will be delayed.
  2. It is tough to track the communication in this way.
  3. We think, it is not wise to show eveyone your website's URL and other sensitive information.

Support Hours and Time

We give reply to our support emails within 24-48 hours.

Advanced or Modification Support

If you request us help, which only needs us to write a couple lines of code, we will assist you in that. But if your query requires to write a lot of codes to be published, we can only provide you guidance and you need to hire a developer if you don't have that skills to implement it yourself.
